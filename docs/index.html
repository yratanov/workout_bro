Easy Deployment

Get Started

Up and running in under a minute.

# docker-compose.yml
services:
  web:
    image: yratanov/workout_bro:latest
    volumes:
      - workout_data:/rails/storage
    ports:
      - "3000:80"
    restart: unless-stopped

  jobs:
    image: yratanov/workout_bro:latest
    command: bin/jobs
    volumes:
      - workout_data:/rails/storage
    restart: unless-stopped
    depends_on:
      - web

volumes:
  workout_data:
1

Create the file

Save the above as docker-compose.yml

2

Start the app

Run docker compose up -d

3

Open in browser

Go to http://localhost:3000
